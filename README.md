# Poderes-Bobos
Sistema em python que armazena nomes de super poderes imaginados, suas descrições e comentários feitos acerca dele.
Um jogo de imaginação simples e divertido!
--------------------------------------------------------------------------------------------------------------------
Regras ao adicionar:
- Deve ser um poder:
  -Bobo;
  -Específico;
  -Não prejudicial ao usuário e outras pessoas.
